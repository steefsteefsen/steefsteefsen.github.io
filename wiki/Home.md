# Welcome to the Wiki

This is the knowledge base for **steefsteefsen.github.io** — the personal homepage of Stefan-Olav Hüllinghorst.

The site is hosted on **GitHub Pages** and mirrored to **GitLab** for CI/CD. Every time a new version lands on `main`, GitLab automatically runs tests and security scans.

## Quick facts

- **Live URL:** https://steefsteefsen.github.io
- **GitHub repo:** https://github.com/steefsteefsen/steefsteefsen.github.io
- **GitLab repo:** https://gitlab.com/steefSteefsen/steefsteefsen.github.io
- **Tech stack:** Plain HTML, CSS, JavaScript — no build tools
- **Aesthetic:** Earthy/natural (cream, clay, moss, bark)
- **Fonts:** Cormorant Garamond (headings) + Jost (body) via Google Fonts
